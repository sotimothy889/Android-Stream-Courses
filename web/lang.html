The Languages

Web development uses 3 major languages for frontend. HTML, CSS and JS. In this introduction to Web development, we will be going over HTML and CSS briefly.

What is HTML?

HTML stands for Hypertext Markup Language, and is a markup language for websites. The site you are seeing now is in HTML. We will be going over HTML in the next page.

What is CSS?

CSS stands for Cascading Style Sheets, and is used for styling websites. We will be going over CSS in later pages.

What is JS?

JS stands for JavaScript, and is used to make interactive elemnts in websites. It is also a server-side programming language. We will not be going over JS as it is not necessary to make basic pages.
