CSS Syntax

CSS is used for styling elements. CSS syntax goes around adding properties to elements, which are identified using selectors. Learn more about selectors here.

For instance, if I would like to make the text colour of all p elements white:

p {
  color: white;
  }

For adding properties, you simply put them in the braces seperated by lines and semicolons. There are many properties; we will not be going over them in this course. See herefor a complete list of properties.
