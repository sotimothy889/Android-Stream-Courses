Tags and Elements in HTML

HTML is focused around the concept of tags and elements. Elements are items that you can see. For example, this text that you are reading is a paragraph element. The browser renders the elements to form a complete page.

Tags are used to form an element. Elements that contain content have both a start tag and an end tag, while those that have no content do not have an end tag.

Example of an element:

<p>I am a paragraph element.</p>

Attributes in elements

Attributes can be added to an element to change how it looks or behaves. It is expressed as name-value pairs: name="value". For example, href is an attribute in anchor tags that specifies the URL of the page the link goes to.

<a href="https://google.com">I am a link that goes to google.com</a>

There are many tags in HTML. Go to W3Schools or MDN Web docs to learn more.
